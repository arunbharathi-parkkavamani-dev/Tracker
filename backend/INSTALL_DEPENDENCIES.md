# Required Dependencies Installation

## Backend Dependencies
```bash
cd backend
npm install country-state-city
```

## Frontend Dependencies  
```bash
cd frontend
# No additional dependencies needed - using existing @mui/material
```

## Mobile Dependencies
```bash
cd App
# No additional dependencies needed - using existing components
```

After installation, the country-state-city hierarchy will work with:
- Country selection first
- State selection depends on country
- City selection depends on state
- Automatic clearing of dependent fields when parent changes
