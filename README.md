# Tracker
**Logimax Organization Full-Fledged HR Admin Panel**

A scalable and modular Employee Management System designed for web (React), mobile (React Native), and document-sharing (Next.js) frontends, with a robust backend and live updates support (Socket.io ready).

The system is divided into **five phases**:

### Phase 1: Core HR & Employee Management
- Role-based dynamic routing and access
- Employee creation & update (HR controlled)
- Employee profile ("My Self" page)
- Daily Check-in / Check-out (Attendance)
- Leave & Permission Requests (Approval / Rejection)
- Daily Activity Updates
- Monthly Summary Reports (Attendance, Leaves, Tasks)

### Phase 2: Expense Tracker
- Employee expense submissions
- Manager approval workflows
- Expense reporting dashboards

### Phase 3: Performance & Reporting
- Employee performance tracking
- Weekly / Monthly reports
- Analytics dashboards for HR / Management
- Notifications and alerts for deadlines and approvals

### Phase 4: Payroll Management
- Integration with payroll system
- Salary adjustments based on attendance & leave
- Dedicated Next.js routes for salary slips and offer letters

### Phase 5: Future-proof Extensions
- Real-time updates via Socket.io
- API analytics & logging
- AI-assisted insights (optional)
